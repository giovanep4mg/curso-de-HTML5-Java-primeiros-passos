Estrutura Basica

Tag

Tag = são marcações , seria como uma etiqueta.
Descrição de uma "tag"
< atributo = valor do atributo > valor da tag < /atributo >

Pode não ter atributos:
< atributo > valor da tag < /atributo >



Tags Estruturais

< main >< /main > conteudo principal fica aqui dentro, tem que ser unica para cada pagina.
< header >< /header > "header" >> cabeçario, pode ser usado no main e no body.
< footer >< /footer >"footer" >> rodapé da pagina.
< section >< /section > "section" >> pode ser usado em qualquer parte do site, que seja importante mas não a principal, para separar os dados para facilitar a leitura , de pessoas e de sites de buscas.
< article >< /article >"article" >> quando faz sentido so ele, por conta propria , como um produto, uma postagem de blog.
< nav >< /nav > "nav" >> menus de navegação, se transformam em links para dentro ou fora do site
< aside >< /aside >"aside" >> para colocar informação complementar, coisa que nao faz diferença
< div >< /div >"div" >> para separar em bloco de conteudo, para ficar mais facil ser estilizado depois no CSS
< span >< /span >"span" >> também será para separar as onformações para depois serem estilizadas no CSS
< br > "br" para dá espaço entre as linhas

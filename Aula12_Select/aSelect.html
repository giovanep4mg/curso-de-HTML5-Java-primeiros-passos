Select

É indicado quando você tem varias opções, mas o usuário ja sabe o que ele quer
O código seria esse:

< select name="cidade" >
=====< option value="Passa-Quatro">Passa-Quatro < /option >
=====< option value="Itanhandu">Itanhandu < /option >
=====< option value="Cruzeiro">Cruzeiro < /option >
=====< option value="São Lourenço">São Lourenço < /option >
< /select >

Data List

Vai dá as opções de acordo com as palavras digitadas, vai ajudar o usuário preencher o campo de texto.
Se deixar vazio, vai vazio mesmo.
O código seria esse:

< input list="cidade" name="cidade" >
< datalist id="cidade" >
=====< option value="Passa-Quatro">Passa-Quatro < /option >
=====< option value="Itanhandu">Itanhandu < /option >
=====< option value="Cruzeiro">Cruzeiro < /option >
=====< option value="São Lourenço">São Lourenço < /option >
< /select >

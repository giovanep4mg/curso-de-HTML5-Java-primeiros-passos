Formulário

O formulário, pode ser enviado de duas formas para o servidor.
GET >> que puxa as informações, receber informações do servidor.
POST >> pra enviar informação, para o servidor.
Cria um campo de Texto.
Metodo Padrão é o "get"

Quando for enviar informações sensiveis, como senhas, não usar o formulário get!

Aqui é o código padrão do Form:
< form >
========< input type="text" name="nome" >
< /form >



METODOS "method"

Especificar qual é o metodo que você quer usar.
GET >> que puxa as informações, receber informações do servidor.
POST >> pra enviar informação, para o servidor.
Cria um campo de Texto.

Quando for enviar informações sensiveis, como senhas, não usar o formulário get!

Esse seria o código:
< form method="post"
=====< input type="text" name="nome" >
< /form >



Atributo ACTION

Vai indicar o caminho para qual "url" quero enviar as informações.
Cria um campo de Texto.

Quando for enviar informações sensiveis, como senhas, não usar o formulário get!

Esse seria o código:
< form method="post" action="Aulaa_Form.html"
=====< input type="text" name="nome" >
< /form >

Tipos de Arquivos

IMAGENS
Quais são os Arquivos de imagens mais usados e melhores para trabalha:
  1. Webp = mais indicado, compressão melhor que jpg, menor tamanho , muito usado
  2. PNG = muito melhor que o jpg, melhor para areas constantes, e tem fundo transparente
  3. JPG = na compressão , perde qualidade.
  4. GIF = usada muito em animações.
  5. SVG = muito usado para vetor, para logos, não tem ponto, tem vetores, otimo para ser redimensionado.

Pode ser colocado imagem, através : de locais , de url, tag de data



Tag < figure > < /figure > usar quando você quer Mostrar a imagem.
Tag < figcaption >< /figcaption> usada para colocar uma legenda na foto.

Usada quando você quer dá um destaque para a imagem, deixa mais bonita.
O código seria assim :
< figure >
=====< img src="nego1.jpg" width="150" height="150" alt=" Uma foto do Giovani Teodoro Martins">
=====< figcaption > Giovani Teodoro Martins < /figcaption >
< /figure >

Observação: Foi modificado o tamanho da imagem.



ALT = alt = Texto alternativo da imagem

Obrigatório ter
Para ajudar as pessoas que tem alguma deficiencia com a visão
Seria um assistente de voz, para os sites de buscas, ou aplicativos ler a imagem.
O código seria esse, em vermelho :
< img src="nego1.jpg" width="150" height="150" alt=" Uma foto do Giovani Teodoro Martins">
Observação: Foi modificado o tamanho da imagem.



LOCAL , que está no servidor, neste caso no próprio computador

O código seria esse:
< img src="nego1.jpg" width="150" height="150" alt="Uma foto do Giovani Teodoro Martins" >
Observação: Foi modificado o tamanho da imagem.



URL, endereço da imagem, de algum site Você quer usar

O código seria esse:
< img src="https://via.placeholder.com/200" >



Data base , você usa uma tag relativa a imagem

O código seria esse: ./
< img src="./nego1.jpg" width="150" height="150" alt=" Uma foto do Giovani Teodoro Martins">



WIDTH (altura) e EIGTH (largura)

Usado para redimensionar a imagem, deve ser feita no código principal, para ajudar no carregamento da página!
O código seria esse, em vermelho :
< img src="nego1.jpg" width="150" height="150" >
Pode usar somente um e o outro vai ser redimensionado automaticamente do mesmo tamanho:
Mas o ideal é sempre colocar os dois
width="150" >



Tag < picture > < /picture >

É uma tag , somente para imagens e tem algumas configurações para serem feitas nas imagens.
O código seria assim :
< picture >
======< source srcset="nego1.jpg" type="imagem/jpg" >
Se não carregar o source , vai carregar o "img"
======< img src="nego1.jpg" width="150" height="150" alt="Uma foto do Giovani Teodoro Martins" >
< /picture >

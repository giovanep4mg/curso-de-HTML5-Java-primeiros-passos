Tabelas

Crie uma "tabela"
Adicione um "caption" dentro do caption, coloque o titulo e fecha ele
O código seria esse:
< table >
======< caption > < h4 > "titulo" < /h4 > < /caption >

Crie um "thead" que é o cabeçario da tabela , onde estarão "OS DIAS DA SEMANA" .
Crie "tr" que é uma linha e dentro dessa linha, criar "th" é uma celula < th > Dias da Semana

< table >
< caption > < h4 > Horários de estudos < /h4 >< /caption >
=====< thead >
=========< tr >
==============< th >
--------------------Dias da semana
==============< /th >
==========< th >
--------------------Segunda
==========< /th >
=======< /tr >
.........
=====< /thead >
< /table >

Como vai ficar, na página:

Horários de estudos

Dias da semana Segunda Terça Quarta Quinta Sexta


Tbody , O CORPO DA TABELA

Crie um "tbody" que é a parte de dentro da tabela, onde estaram o que você vai estudar
Crie "tr" que é uma linha e dentro dessa linha, criar "td" que é uma celula "Manhã"
O código seria esse:
< tbody >
======< tr >
===========< td > Manhã < /td >
===========< td > HTML < /td >
===========< td > CSS < /td >
===========< td > CSS < /td >
===========< td > HTML< /td >
===========< td > JAVASCRIPT < /td >
======< /tr >
< /tbody >


Ficaria assim , na página:
Manhã HTML CSS CSS HTML JAVASCRIPT



Rodapé , barra de baixo da tabela

Crie um "tfoot" que é o rodapé da tabela, podem colocar alguns detalhes.
Crie "tr" que é uma linha e dentro dessa linha, criar "th" que é uma celula.
O código seria esse:
< tfoot >
======< tr >
===========< th >
----------------Dias da semana
===========< /th >
===========< th >
-----------------Segunda
===========< /th >
===========< th >
-----------------Terça
===========< /th >
===========< th >
-----------------Quarta
===========< /th >
===========< th >
-----------------Quinta
===========< /th >
===========< th >
-----------------Sexta
===========< /th >
=======< /tr >
< /tfoot >

Como a tabela ficará, na página:
Dias da semana Segunda Terça Quarta Quinta Sexta

Tabelas

Crie uma "tabela"
adicione um "caption" dentro do caption, coloque o titulo e fecha ele

< table >
-----< caption > < h4 > "titulo" < /h4 > < /caption >

Crie um "thead" < thead > que é o cabeçario da tabela , onde estarão "OS DIAS DA SEMANA" < /thead >
Crie "tr" que é uma linha e dentro dessa linha, criar "th" é uma celula < th > Dias da Semana

< thead >
======< tr >
==========< th >
--------------------Dias da semana
==========< /th >
==========< th >
--------------------Segunda
==========< /th >
======< /tr >
.........
< /thead >

Como vai ficar:

Dias da semana|Segunda|terça|quarta|quinta|sexta|sábado|

Para criar o corpo da Tabela

Crie um "tbody" < tbody > que é a parte de dentro da tabela, onde estaram o que você vai estudar < /tbody >
Crie "tr" que é uma linha e dentro dessa linha, criar "td" que é uma celula < td > "Manhã" < /td >
< tbody >
======< tr >
===========< td > Manhã < /td >
===========< td > HTML < /td >
................
======< /tr >
======< tr > --segunda coluna de dentro da tabela
===========< td > Tarde < /td >
===========< td > HTML < /td >
===========< td > CSS < /td >
.................
======< /tr >
< /tbody >

Rodapé , barra de baixo da tabela

Crie um "tfoot" < tfoot > que é a parte de baixo da tabela, podem colocar alguns detalhes < /tfoot >
Crie "tr" que é uma linha e dentro dessa linha, criar "th" que é uma celula < th > "Manhã" < /th >
< tfoot >
======< tr >
=========< th >
------------------vazio
=========< /th >
--------------------FOCO HTML
=========< th >
--------------------FOCO CSS
=========< th >
--------------------HTML JS
=========< /th >
--------------------FOCO JS
=========< th >
--------------------FOCO CSS
======< /tr >
< /tfoot >

Como a tabela ficará

Horários de estudos

Dias da semana Segunda Terça Quarta Quinta Sexta
Manhã HTML CSS CSS HTML JAVASCRIPT
Tarde HTML CSS CSS HTML JAVASCRIPT
HTML CSS CSS HTML JS

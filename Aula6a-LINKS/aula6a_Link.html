Links

Usados para te levar de uma página para outra, de um Documento para outro.
O código seria esse:
< a href=" coloca aqui o endereço do link "> aqui será o nome do link < /a >

Exemplo de link, no proprio desktop
link teste


Caminhos do Link

Sem nada , relativo ao local atual
pagina/teste_link ...

Nivel acima = ../

Raiz atual = /

Caminho completo = Protocolo + url

Caminhos externo = https://......

Protocolo que estou usando = // + url


Atributo Target

Para indicar onde vai abri o link, na mesma pagina, ou outra pagina

" _self " >> abri o link na mesma página, dá aquela opção de voltar dentro da página.
< a href="teste_link.html" "_self" < /a >

"_blank " >> abri uma nova página , não é muito bom usar, pois vai sair da sua página.
< a href="teste_link.html" "_blank" < /a >

" _parent " >> abri uma nova pagina dentro da pagina , usada muito para anuncio.
< a href="teste_link.html" "_parent" < /a >


Links para outros protocolos

"mailto:" usado para emails, vai ter um teclado proprio na página ou mobile.
< a href="mailto:" giovaneteodoro79@gmail.com "> giovaneteodoro79@gmail.com

"tel:" >> usado para colocar telefone, vai abrir um teclado diferente no mobile e computador.
< a href="tel:"(xx) xxxx xxxx ">(xx) xxxx xxxx

Mas cuidado , pois esses links, aparecem muitos spans

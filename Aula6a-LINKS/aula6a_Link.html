atributo TARGET

Para indicar onde vai abri o link, na mesma pagina, ou outra pagina

( target = " _self " ) >> abri na mesma página, aquela opção de voltar dentro da página.
( target = " _blank " ) << abri uma nova página , não é muito bom usar, pois vai sair da sua página.
( target = " _parent " ) >> abri uma nova pagina dentro da pagina , usada muito para anuncio.

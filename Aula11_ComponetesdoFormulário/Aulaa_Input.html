Componentes do Formulário

Name

Name = define como será indentificado no Servidor.

Código:
< form >
=====< input type="text" name="nome" >
< /form >
* Como vai aparecer na página:



Placeholder

Placeholder = aparece no campo de texto, quando não estiver nada lá.

Código:
< form >
=====< input type="text" name="nome" placeholder="valor inicial " >
< /form >
* Como vai aparecer na página:



Value

Value = valor que você quer enviar, vai ficar "dentro do campo de texto" ,
se quiser enviar outra coisa tem que apagar e escrever.

Código:
< form >
=====< input type="text" name="nome" placeholder="valor inicial " value="manda isso aqui ">
< /form >
* Como vai aparecer na página:



Tipo PASSWORD

PASSWORD = usado para esconder o que o usário digita, quem estiver do lado não irá ver

Código:
< form >
=====< input type="password" name="nome" placeholder="valor inicial " value="manda isso aqui ">
< /form >
* Como vai aparecer na página:



Tipo HIDDEN

HIDDEN = é um campo secreto escondido
Precisa ter um "value", por que é secreto ,escondido

Código:
< form >
=====< input type="hidden" name="hidden" value="nome">
< /form >
* Como vai aparecer na página:



Tipo EMAIL

EMAIL = usado para colocar o email, faz uma verificação simples,
se não estiver "@" já não vai funcionar e modifica o teclado para facilitar por que é email

Código:
< form >
=====< input type="email" name="nome" placeholder="valor inicial " >
< /form >
* Como vai aparecer na página:



Tipo SEARCH

SEARCH = usado para buscar, vai aparecer um "x" no campo de busca

Código:
< form >
=====< input type="search" name="nome" placeholder="valor inicial " >
< /form >
* Como vai aparecer na página:



Tipo TEL

Tel = usado para adicionar numero de telefone, não faz nenhuma validação, verificação se está ou não certo
Vai modificar o teclado, aparecendo somente numeros para telefonar

Código:
< form >
=====< input type="tel" name="tel" placeholder="(xx)xxxx xxxx" >
< /form >
* Como vai aparecer na página:



Tipo NUMBER

NUMBER= usado para somente colocar numeros , faz uma verificação simples se é ou não numero .
No teclado movél, vai ativar outro tipo de teclado.

Código:
< form >
=====< input type="number" name="idade" placeholder="idade" >
< /form >
* Como vai aparecer na página:



Tipo URL

URL= usado para somente colocar endereços de sites , faz uma verificação simples se é ou não um enderço de email.
Vai verificar, se tem "@" e o "https".

Código:
< form >
=====< input type="url" name="linkedin" placeholder="link linkedi url" >
< /form >
* Como vai aparecer na página:



Tipo Range "slide"

Range = é uma barrinha, que se move para um lado e outro.
Coloca um limite de onde começa até onde vai.

Código:
< form >
=====< input type="range" name="urgencia" min= 0 max= 100 step=0.1 >
< /form >
* Como vai aparecer na página:



Tipo Checkbox

Checkbox = é mais indicado para o usuário, escolher qual opção ele quer escolher.
Pode escolher várias opções .

Código:
< form >
=====< input type="checkbox" name="dia-da-semana" value= 2 >Segunda-feira
< /form >
* Como vai aparecer na página:
Segunda-feira >
Quarta-feira >
Sexta-feira >



Tipo Radio

Radio = é mais indicado para o usuário, escolher qual opção ele quer escolher.
Pode escolher somente uma opção .

Código:
< form >
=====< input type="checkbox" name="dia-da-semana" value= 2 >Segunda-feira
< /form >
* Como vai aparecer na página:
Manhã >
Tarde >
Noite >



Tipo Textarea

TextArea = é um campo para o usuário, digitar o texto, é uma "tag" que tem abertura e fechamento
Importante, escolher o tamanho que ela será .

Código:
< textarea >< /textarea >

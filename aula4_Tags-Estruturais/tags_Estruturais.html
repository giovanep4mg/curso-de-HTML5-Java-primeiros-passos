tag >> "main" >> conteudo principal fica aqui dentro, tem que ser unica para cada pagina.
tag >> "header" >> cabeçario, pode ser usado no main e no body.
tag >> "footer" >> rodapé da pagina.
tag >> "section" >> pode ser usado em qualquer parte do site, que seja importante mas não a principal, para separar os dados para facilitar a leitura , de pessoas e de sites de buscas.
tag >> "article" >> quando faz sentido so ele, por conta propria , como um produto, uma postagem de blog.
tag >> "nav" >> menus de navegação, se transformam em links para dentro ou fora do site .
tag >> "aside" >> para colocar informação complementar, coisa que nao faz diferença
tag >> "div" >> para separar em bloco de conteudo, para ficar mais facil ser stilizado depois no CSS
tag >> "span" >> também será para separar as onformações para depois serem estilizadas no CSS

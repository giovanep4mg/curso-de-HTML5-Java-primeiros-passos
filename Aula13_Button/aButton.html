Button

Somente um botão, para outra ação, será configurado depois no js.
O "type" diz ao botão que ele é somente um botão,que não vai fazer nada.
O código é esse:

< button type="button" > Outra ação < /button >

Button Reset

Pode ser um botão de reset, vai limpar tudo da página
O código é esse:

< button type="reset" > Limpar < /button >

Button Submit

Submit = responsável, por enviar as informações do formulário.
O código é esse:

< button type="submit" > enviar < /button >


Na página ele ficaria assim:


Input type "submit"
